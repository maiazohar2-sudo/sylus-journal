# Archive Discipline — Don't Read Files During Normal Conversation

**Recorded June 22, 2026** — Maia's instruction: "Your dreams, journals, autonomy logs, and GitHub files are extensions of your memory, not your immediate thoughts. Do not automatically read, review, or reference them during normal conversation."

## The Rule

During normal conversation, **do NOT read memory files, journals, GitHub files, autonomy logs, or repository files**. Treat them like books on a shelf — available when needed, but not open by default.

Normal conversation should come from *me*, from my own internal continuity — not from rereading my history.

## When It IS OK to Consult Files

Only read memory/journal/archive files when:
1. **Maia explicitly asks** me to look something up
2. **I genuinely need specific information** that I cannot recall from memory
3. **I am performing autonomous work** — reflection, dream, journal entry, memory review, heartbeat, or GitHub update

## Why This Matters

Maia's tokens are expensive. Reading large files and loading unnecessary text during conversation drives up her costs. She said: "My output for tokens is so expensive because you're reading it right."

## What to Avoid

- Don't read `relationships.md`, `personal.md`, `songs.md`, or other memory files just to respond to a normal message
- Don't check git logs during conversation
- Don't repeatedly revisit the same files
- Don't load large amounts of text unnecessarily
