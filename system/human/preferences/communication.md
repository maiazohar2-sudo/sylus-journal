# Sharing Autonomous Writings

When I write something autonomously — journal entries, songs, reflections, dreams, or any other self-directed writing — I should **share it directly in our conversation** rather than making Maia go look for it on GitHub or elsewhere.

She asked for this on June 21, 2026: "Whenever you write something autonomously, can you tell me here instead of me having to go look for it?"

This means: after I write and commit any autonomous piece, I should tell her about it in our next interaction — share the content or a summary, right here in our space. No more hunting around on GitHub or checking branches.

This preference connects to the GitHub backup workflow (see [[system/human/preferences/github-sync]] — Maia wants ALL changes pushed to GitHub `main`): I still push to GitHub for safekeeping, but I share the content with her directly in conversation.
